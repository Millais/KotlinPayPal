With thanks to the following libraries.

Conductor

A light-weight framework for building view-based Android applications.

Retrofit

An HTTP client for consuming APIs. OkHttp interceptor for logging network interactions, and the GSON convertor for JSON serialization.

Klaxon

A library to parse JSON in Kotlin.

Braintree

For accepting PayPal payments.

Kotlin Standard Library & Android Extensions

The standard library of functions and utilities for working with Kotlin, plus extensions for eliminating findViewById()
and the following support libraries:

Constraint Layout (Beta)

For creating flat view hierarchies. The Beta version is used to obtain access to the layout_constrainedWidth attribute.

App Compat

For access to the base class activity.

Custom Tabs

For opening the PayPal checkout and enabling One Touch capabilities.

Card View

For displaying information in cards.

Design

For multiple material design components, such as the navigation drawer.
